- ### 实现垂直连接线
```html
<div class="fu">
  <div class="zi-line1"></div>
  <div class="zi-spot"></div>
  <div class="zi-line2"></div>
</div>
```
```css
.fu{
display:flex
height:100%/auto
-webkit-box-orient:verticle
flex-direction:column
align-item:center
}
.line1{
//定高度
}
.spot{//设圆点}
.line2{
flex:1 //自适应剩下高度，父盒子大小
width:1px;
}
```
