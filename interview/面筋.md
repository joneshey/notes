## ES6
1. let和var区别: 
不存在变量提升；
```
console.log(c)  //undefined，如果是let会直接报错
var c = 1;
=> var c;
c = 1
```  
同一个作用域不能重复定义；  
  
有严格的作用域，var是函数作用域，let是块级作用域  

* const如果是常量不能改变，但如果是对象则可以修改内部属性。  

2.  箭头函数
v为参数，v+1返回值  
`var a = v => v+1`  
无参数  
`var a = () => 2`  
*this执行定义的主体而不是调用的主体。

3. set、map结构函数
set 类似数组，成员唯一  （转变数组[...new Set()]）   
map 键可以是对象  可以使用for of会把原型也进行遍历，且要注意兼容性（for in要数组）

4. 模块字符串
反义字符串：反单引号``  
<a>${val}</a> //val为变量
